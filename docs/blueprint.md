# **App Name**: HoopsCoach

## Core Features:

- Firebase Authentication: Email/password login, sign-up, forgot password, onboarding slides to manage user authentication.
- Role-Based Content: Control content display based on user roles, offering a tailored experience.
- Workout Tracking: Track user progress, workouts completed, and statistics; record user inputs to Firestore.
- Profile Management: Manage user profiles, including photo uploads, editable attributes (age, height, weight, position), and progress tracking.
- Workout Library: Access a library of basketball-specific workouts, categorized by skills (shooting, ball handling, defense, conditioning, vertical jump), with detail pages that display the title, difficulty, duration and video urls.
- Meal Planner: Plan meals for athletes based on categories, with details that include photo, ingredients, steps, and macro summaries. Save plans for the athlete and track progress. Suggest changes using AI if progress isn't meeting target goals; use this as a tool.
- BMI Calculator: Estimates the user's body mass index from the user's inputs for height and weight, suggesting training adjustments accordingly.

## Style Guidelines:

- Primary color: Vibrant orange (#FF8C00) for energy and basketball theme.
- Background color: Dark gray (#333333) for a modern, sporty look.
- Accent color: Light gray (#A9A9A9) for contrast and readability.
- Body and headline font: 'Inter', sans-serif, a neutral modern font that works for both headings and body copy.
- Use basketball-themed icons and minimalist designs for a sporty and clean look.
- Bottom navigation for main tabs, cards for workouts, and clear information hierarchy.
- Subtle transitions and animations for user interactions to enhance engagement.
